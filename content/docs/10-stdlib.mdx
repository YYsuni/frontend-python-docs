## 1. 操作系统接口（os / shutil）

- **os**：底层系统操作，**禁止 from os import \***（避免覆盖内置 open）
- **shutil**：文件/目录高级操作，更易用

```python
import os
import shutil

# 当前目录
os.getcwd()
# 切换目录
os.chdir('/path')
# 执行系统命令
os.system('mkdir demo')

# 复制文件
shutil.copyfile('src.txt', 'dst.txt')
# 移动/重命名
shutil.move('src', 'dst')
```

## 2. 文件通配符（glob）

快速按规则匹配文件列表

```python
import glob
# 匹配所有 .py 文件
print(glob.glob('*.py'))
```

## 3. 命令行参数（sys / argparse）

- **sys.argv**：简单参数列表
- **argparse**：专业命令行解析

```python
# sys.argv 用法
import sys
print(sys.argv)  # 运行: python test.py a b → ['test.py','a','b']

# argparse 用法
import argparse
parser = argparse.ArgumentParser(description='显示文件前N行')
parser.add_argument('files', nargs='+')
parser.add_argument('-l', '--lines', type=int, default=10)
args = parser.parse_args()
print(args.files, args.lines)
```

## 4. 错误输出与程序终止（sys）

```python
import sys
# 标准错误输出（不被重定向覆盖）
sys.stderr.write('错误信息\n')
# 终止程序
sys.exit()
```

## 5. 字符串匹配（re）

复杂匹配用正则，简单操作用字符串方法

```python
import re
# 查找以 f 开头的单词
re.findall(r'\bf[a-z]*', 'foot fell fastest')
# 去重连续单词
re.sub(r'(\b[a-z]+) \1', r'\1', 'the the cat')

# 简单替换优先用字符串方法
'tea for too'.replace('too', 'two')
```

## 6. 数学与随机（math / random / statistics）

```python
import math
import random
import statistics

# 数学计算
math.cos(math.pi / 4)
math.log(1024, 2)

# 随机
random.choice(['apple', 'pear'])
random.sample(range(100), 5)
random.randint(1, 10)

# 统计
data = [1, 2, 3, 4, 5]
statistics.mean(data)   # 均值
statistics.median(data) # 中位数
```

## 7. 网络请求（urllib）

极简获取网页内容

```python
from urllib.request import urlopen
with urlopen('https://httpbin.org/get') as resp:
    print(resp.read().decode())
```

## 8. 日期时间（datetime）

```python
from datetime import date
now = date.today()
# 格式化
print(now.strftime('%Y-%m-%d %A'))
# 日期计算
birth = date(1990, 1, 1)
print((now - birth).days)
```

## 10. 性能测试（timeit）

对比代码执行速度

```python
from timeit import Timer
# 对比变量交换两种写法
Timer('t=a;a=b;b=t', 'a=1;b=2').timeit()
Timer('a,b = b,a', 'a=1;b=2').timeit()
```

## 11. 代码质量（doctest / unittest）

- **doctest**：文档字符串里写测试
- **unittest**：大型项目专业测试

```python
# doctest
def avg(x):
    """求平均 >>> avg([1,2,3]) 2.0 """
    return sum(x)/len(x)
import doctest; doctest.testmod()

# unittest
import unittest
class TestAvg(unittest.TestCase):
    def test_avg(self):
        self.assertEqual(avg([1,2,3]), 2.0)
unittest.main()
```

## 12. 数据处理（自带电池）

- **json**：JSON 序列化
- **csv**：表格文件读写
- **sqlite3**：轻量数据库
- **xml**：XML 解析

```python
import json
data = {'name': 'py', 'age': 20}
# 转 JSON 字符串
json_str = json.dumps(data)
# 转回对象
obj = json.loads(json_str)
```
