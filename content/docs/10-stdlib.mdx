## 10.1 速查心智图

| 你要做     | 先用标准库           | 常见升级             |
| ---------- | -------------------- | -------------------- |
| 路径与文件 | `pathlib` + `shutil` | 云存储 SDK           |
| CLI 参数   | `argparse`           | `click` / `typer`    |
| HTTP       | `urllib`             | `httpx` / `requests` |
| 定时/日期  | `datetime`           | `zoneinfo`（3.9+）   |
| 表格       | `csv`                | `pandas`             |

## 10.2 操作系统接口：`os` / `shutil`

- **`os`**：进程、环境变量；路径优先 `pathlib`。
- **不要** `from os import *`：会盖住内置 `open`。

```python
from pathlib import Path
import shutil

src = Path("src.txt")
dst = Path("demo") / "readme.txt"
dst.parent.mkdir(parents=True, exist_ok=True)
shutil.copyfile(src, dst)
print(Path.cwd())
```

## 10.3 通配符：`glob`

```python
import glob
from pathlib import Path

print(glob.glob("*.py"))
print([p.name for p in Path(".").rglob("*.py")])
```

## 10.4 CLI 参数：`argparse`

```python
import argparse

parser = argparse.ArgumentParser(description="打印文件前 N 行")
parser.add_argument("files", nargs="+")
parser.add_argument("-n", "--lines", type=int, default=10)
args = parser.parse_args()
print(args.files, args.lines)
```

## 10.5 标准流与退出：`sys`

```python
import sys

print("正常信息")
print("告警：磁盘将满", file=sys.stderr)
sys.exit(1)  # 退出码非 0，shell/CI 会当作失败
```

## 10.8 极简 HTTP：`urllib`

```python
from urllib.request import urlopen

with urlopen("https://example.com", timeout=5) as resp:
    snippet = resp.read(256)
    print(snippet[:64])
```

> Cookie、重试、连接池：换 `httpx` / `requests`。

## 10.9 日期时间：`datetime`

```python
from datetime import date, datetime, timedelta, timezone

today = date.today()
print(today.strftime("%Y-%m-%d"))

utc_now = datetime.now(timezone.utc)
print(utc_now.isoformat())

print(today - timedelta(days=7))
```

## 10.11 微基准：`timeit`

```python
from timeit import timeit

print(timeit("t=a; a=b; b=t", "a=1; b=2"))
print(timeit("a,b = b,a", "a=1; b=2"))
```

## 10.12 测试：`doctest` / `unittest`

```python
def avg(xs: list[float]) -> float:
    """Return average.

    >>> avg([1, 2, 3])
    2.0
    """
    return sum(xs) / len(xs)


if __name__ == "__main__":
    import doctest

    doctest.testmod()
```

## 10.14 学习顺序建议

1. `pathlib` → 少写字符串路径拼接。
2. `argparse` + `if __name__ == "__main__"` → 可发布 CLI。
3. `datetime` + `json` + `re` → 日常胶水三件套。
