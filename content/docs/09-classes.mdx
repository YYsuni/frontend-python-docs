## 一、核心概念

- **类**：数据 + 功能的封装，创建新类型。
- **实例**：类的具体对象，有自己的属性与方法。
- **self**：方法第一个参数，代表实例本身（约定俗成）。
- **属性查找**：实例 → 类 → 父类，逐级查找。

## 二、命名空间与作用域

- 命名空间：名字→对象的映射（字典实现）。
- 查找顺序：**局部 → 外层闭包 → 全局 → 内置**。
- `global`：声明使用全局变量。
- `nonlocal`：声明使用外层函数变量（非全局）。

### 示例

```python
def scope_test():
    spam = "test spam"
    def do_local():
        spam = "local spam"  # 只改局部
    def do_nonlocal():
        nonlocal spam       # 改外层函数
        spam = "nonlocal spam"
    def do_global():
        global spam         # 改全局
        spam = "global spam"
    do_local()
    print(spam)  # test spam
    do_nonlocal()
    print(spam)  # nonlocal spam
    do_global()
    print(spam)  # nonlocal spam
scope_test()
print(spam)      # global spam
```

## 三、类基础语法

### 1. 定义与实例化

```python
class MyClass:
    # 类属性（所有实例共享）
    i = 12345
    # 构造方法
    def __init__(self, name):
        self.name = name  # 实例属性（各实例独立）
    # 实例方法
    def say_hello(self):
        return f"Hello, {self.name}"

# 实例化
obj = MyClass("Python")
print(obj.name)       # Python
print(obj.say_hello()) # Hello, Python
```

### 2. 类变量 vs 实例变量

- **类变量**：全实例共享。
- **实例变量**：每个实例独立。

```python
class Dog:
    # 正确：类变量（共享）
    kind = "canine"
    def __init__(self, name):
        # 正确：实例变量（独立）
        self.name = name
        self.tricks = []

    def add_trick(self, trick):
        self.tricks.append(trick)

d1 = Dog("Fido")
d2 = Dog("Buddy")
d1.add_trick("roll over")
print(d1.tricks)  # ['roll over']
print(d2.tricks)  # []
```

## 四、方法

- **实例方法**：第一个参数 `self`，调用时自动传入实例。
- **调用等价**：`obj.method()` = `Class.method(obj)`。

```python
class Bag:
    def __init__(self):
        self.data = []
    def add(self, x):
        self.data.append(x)
    def add_twice(self, x):
        self.add(x)
        self.add(x)

bag = Bag()
bag.add_twice(5)
print(bag.data)  # [5,5]
```

## 五、继承

### 1. 单继承

```python
class Animal:
    def speak(self):
        return "..."

class Cat(Animal):
    # 重写方法
    def speak(self):
        return "Meow"

cat = Cat()
print(cat.speak())  # Meow
```

### 2. 调用父类方法

```python
class Animal:
    def __init__(self, name):
        self.name = name

class Dog(Animal):
    def __init__(self, name, breed):
        # 调用父类构造
        super().__init__(name)
        self.breed = breed

dog = Dog("Buddy", "Labrador")
print(dog.name, dog.breed)
```

### 3. 多重继承

搜索顺序：**深度优先、从左到右**，不重复搜索。

```python
class A: pass
class B: pass
class C(A, B): pass
```

### 4. 类型判断

```python
isinstance(obj, Cat)     # 是否为类/子类实例
issubclass(Cat, Animal)  # 是否为子类
```

## 六、私有变量

Python 无真正私有，靠**名称改写**模拟。

- `_var`：约定私有，外部建议不访问。
- `__var`：自动改写为 `_ClassName__var`。

```python
class Mapping:
    def __init__(self, iterable):
        self.items = []
        self.__update(iterable)
    def update(self, iterable):
        self.items.extend(iterable)
    __update = update  # 内部私有副本

m = Mapping([1,2,3])
# m.__update  报错，无法直接访问
```

## 七、迭代器与生成器

### 1. 迭代器

实现 `__iter__` + `__next__`。

```python
class Reverse:
    def __init__(self, data):
        self.data = data
        self.index = len(data)
    def __iter__(self):
        return self
    def __next__(self):
        if self.index == 0:
            raise StopIteration
        self.index -= 1
        return self.data[self.index]

for c in Reverse("spam"):
    print(c)  # m a p s
```

### 2. 生成器（更简洁）

用 `yield`，自动实现迭代器协议。

```python
def reverse(data):
    for i in range(len(data)-1, -1, -1):
        yield data[i]

print(list(reverse("golf")))  # ['f','l','o','g']
```

### 3. 生成器表达式

```python
sum(i*i for i in range(10))        # 平方和
sum(x*y for x,y in zip([10,20],[1,2]))  # 点积
```

## 八、速记要点

1. 构造方法固定叫 `__init__(self, ...)`。
2. 方法必须写 `self`，类似 JS `this`。
3. 类变量共享，实例变量隔离；可变容器务必放实例。
4. 继承用 `class 子类(父类)`，重写覆盖，`super()` 调用父类。
5. 私有靠约定与名称改写，无强制访问控制。
6. 迭代/遍历优先用生成器，代码更短。
