## 7. 输出格式化（4 种方式）

### 7.1.1 f-string（推荐）

- 字符串前加 `f`/`F`，`{变量/表达式}` 直接嵌入
- 支持格式控制、对齐、`!r`/`!s`/`!a`、`=` 自说明

```python
# 基础
name, age = "Alice", 25
print(f"{name} is {age}")

# 精度与对齐
import math
print(f"π ≈ {math.pi:.3f}")

# 对齐打印
table = {"A": 123, "B": 4567}
for k, v in table.items():
    print(f"{k:5} → {v:6d}")

# repr 与自说明
print(f"{name=} {age=}")
```

---

### 7.1.2 str.format()

- 用 `{}` 占位，支持位置/关键字/字典传参
- 适合复杂排版与兼容旧代码

```python
# 位置
print("{0} + {1} = {2}".format(1, 2, 3))

# 关键字
print("{name}: {score}".format(name="Tom", score=99))

# 字典解包
user = {"id": 100, "name": "Amy"}
print("{id} → {name}".format(**user))
```

---

### 7.1.3 手动格式化（字符串方法）

- `ljust()`/`rjust()`/`center()` 对齐
- `zfill()` 左侧补零

```python
for x in range(1, 6):
    print(
        repr(x).rjust(2),
        repr(x*x).rjust(3),
        repr(x**3).rjust(4)
    )

print("123".zfill(6))  # 000123
```

---

### 7.1.4 旧式 % 格式化

- 类似 C printf，不推荐新项目使用

```python
import math
print("pi = %.3f" % math.pi)
```

---

### str() vs repr()

- `str()`：人读友好
- `repr()`：解释器可执行，带引号/转义

```python
s = "hello\n"
print(str(s))   # hello
print(repr(s))  # 'hello\n'
```

## 7. 文件读写（核心）

### 打开文件（with 自动关闭）

```python
# 读文本
with open("test.txt", "r", encoding="utf-8") as f:
    content = f.read()

# 写文本（覆盖）
with open("out.txt", "w", encoding="utf-8") as f:
    f.write("hello\n")

# 追加
with open("log.txt", "a", encoding="utf-8") as f:
    f.write("append\n")
```

### 常用模式

- `r`：只读（默认）
- `w`：只写（覆盖）
- `a`：追加
- `r+`：读写
- `b`：二进制（图片/视频）
- **文本必加 `encoding="utf-8"`**

---

### 文件对象方法

```python
f = open("test.txt", encoding="utf-8")

f.read()       # 全读
f.readline()   # 读一行
for line in f: # 逐行迭代（省内存）
    print(line, end="")

f.write("str") # 写入
f.tell()       # 当前位置
f.seek(0)      # 跳转到开头
f.close()      # 关闭（with 自动做）
```

## 7.2.2 JSON 序列化（存结构化数据）

- 用于字典/列表，跨语言通用
- **不要用 pickle 存不可信数据**

```python
import json

# 写入
data = {"name": "py", "ver": 3.12}
with open("data.json", "w", encoding="utf-8") as f:
    json.dump(data, f, ensure_ascii=False, indent=2)

# 读取
with open("data.json", encoding="utf-8") as f:
    data = json.load(f)
```

## 速记要点

1. **输出优先用 f-string**，简洁高效
2. **文件必用 with**，自动关闭防泄漏
3. **文本用 utf-8**，二进制加 `b`
4. **结构化存 json**，安全通用
