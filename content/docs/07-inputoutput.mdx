## 7.1 输出格式化

### f-string（首选）

```python
import math

name, age = "Alice", 25
print(f"{name} is {age}")
print(f"π ≈ {math.pi:.3f}")

table = {"Sjoerd": 4127, "Jack": 4098, "Dcab": 7678}
for name, phone in table.items():
    print(f"{name:10} → {phone:10d}")

print(f"{name=!s}")  # 自解释调试输出
```

### `str.format`（维护旧代码仍会看到）

```python
print("{} + {} = {}".format(1, 2, 3))
print("{name}: {score}".format(name="Tom", score=99))

user = {"id": 100, "name": "Amy"}
print("{id} → {name}".format(**user))
```

### 手动对齐与补零

```python
for x in range(1, 6):
    print(repr(x).rjust(2), repr(x * x).rjust(3), repr(x**3).rjust(4))

print("12".zfill(5))  # '00012'
```

### 旧式 `%`（了解即可）

```python
import math

print("pi = %.3f" % math.pi)
```

### `str()` vs `repr()`

```python
s = "hello\nworld"
print(str(s))   # 人读友好
print(repr(s))  # 解释器倾向可复现的形式
```

## 7.2 文件读写

**总原则**：文本显式 `encoding="utf-8"`；用 `with` 关文件。Windows 上缺 encoding 有时会踩 **GBK** 默认坑。

```python
from pathlib import Path

text_path = Path("test.txt")
text_path.write_text("hello\n", encoding="utf-8")

print(text_path.read_text(encoding="utf-8"))
```

等价 `open` 写法（需要逐块读或 `seek` 时更常用）：

```python
with open("test.txt", encoding="utf-8") as f:
    content = f.read()

with open("out.txt", "w", encoding="utf-8") as f:
    f.write("hello\n")

with open("log.txt", "a", encoding="utf-8") as f:
    f.write("one more line\n")
```

### 逐行读（省内存）

```python
with open("big.log", encoding="utf-8") as f:
    for line in f:
        line = line.rstrip("\n")
        if line:
            ...  # 在这里处理一行（解析、聚合、写下游）
```

### 二进制模式

```python
png_path = Path("logo.png")
data = png_path.read_bytes()
assert data[:8] == b"\x89PNG\r\n\x1a\n"  # 魔数检查示例
```

## 7.3 JSON

- **适合跨语言**；`datetime` 等**不原生支持**，要自己转字符串或用自定义 `default`。
- **绝不要**用 `pickle` 处理不可信输入（等效执行代码级别风险）。

```python
import json
from datetime import date

payload = {"ok": True, "when": date.today().isoformat()}

print(json.dumps(payload, ensure_ascii=False))

raw = '{"nums": [1, 2, 3]}'
obj = json.loads(raw)
print(sum(obj["nums"]))

# 落地文件：indent 让 diff 可读
with open("data.json", "w", encoding="utf-8") as f:
    json.dump(payload, f, ensure_ascii=False, indent=2)
```
