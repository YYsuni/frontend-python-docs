## 一、列表（list）详解

### 1. 核心方法

| 方法           | 作用                   | 代码示例                       |
| -------------- | ---------------------- | ------------------------------ |
| `append(x)`    | 末尾添加元素           | `a = [1]; a.append(2) → [1,2]` |
| `extend(iter)` | 批量添加可迭代对象     | `a.extend([3,4]) → [1,2,3,4]`  |
| `insert(i,x)`  | 指定索引插入           | `a.insert(0,0) → [0,1,2,3,4]`  |
| `remove(x)`    | 删除第一个匹配值       | `a.remove(2) → [0,1,3,4]`      |
| `pop(i)`       | 删除并返回指定索引元素 | `a.pop() → 4; a → [0,1,3]`     |
| `clear()`      | 清空列表               | `a.clear() → []`               |
| `index(x)`     | 查找第一个匹配值索引   | `[1,2,2].index(2) → 1`         |
| `count(x)`     | 统计元素出现次数       | `[1,2,2].count(2) → 2`         |
| `sort()`       | 原地排序               | `[3,1].sort() → [1,3]`         |
| `reverse()`    | 原地反转               | `[1,2].reverse() → [2,1]`      |
| `copy()`       | 浅拷贝                 | `b = a.copy()`                 |

### 2. 列表应用

- **堆栈（后进先出）**

```python
stack = [1,2]
stack.append(3)  # 入栈
stack.pop()      # 出栈 → 3
```

- **队列（先进先出）**：列表效率低，推荐 `deque`

```python
from collections import deque
q = deque([1,2])
q.append(3)
q.popleft()  # 出队 → 1
```

### 3. 列表推导式

- 基础写法：`[表达式 for 变量 in 可迭代对象 if 条件]`

```python
# 平方列表
squares = [x**2 for x in range(5)]  # [0,1,4,9,16]
# 过滤+转换
[x*2 for x in [-1,0,1] if x>=0]    # [0,2]
# 展平嵌套列表
[[1,2],[3,4]] → [num for elem in [[1,2],[3,4]] for num in elem]
```

### 4. 嵌套列表推导式

```python
# 矩阵转置
matrix = [[1,2],[3,4],[5,6]]
[[row[i] for row in matrix] for i in range(2)]  # [[1,3,5],[2,4,6]]
# 简洁写法：list(zip(*matrix))
```

## 二、del 语句

- 按索引/切片删除，无返回值

```python
a = [1,2,3,4]
del a[0]    # [2,3,4]
del a[1:3]  # [2]
del a[:]    # 清空
```

## 三、元组（tuple）

### 核心特性

- **不可变**：无法修改元素
- **创建**：`()` 或 逗号分隔；单元素必须加逗号

```python
t = (1,2,3)  # 标准创建
t1 = 1,      # 单元素元组 → (1,)
empty = ()   # 空元组
```

- **序列解包**

```python
x, y, z = (1,2,3)  # x=1,y=2,z=3
```

## 四、集合（set）

### 核心特性

- **无序、无重复**
- **创建**：`{}` 或 `set()`；空集合必须用 `set()`

```python
s = {1,2,2,3}  # {1,2,3}
s = set([1,2])
```

### 集合运算

```python
a = {1,2,3}; b = {3,4,5}
a - b   # 差集 → {1,2}
a | b   # 并集 → {1,2,3,4,5}
a & b   # 交集 → {3}
a ^ b   # 对称差集 → {1,2,4,5}
```

## 五、字典（dict）

### 核心特性

- **键值对存储**，键必须不可变（str/int/tuple）
- **创建**

```python
d = {'name':'py', 'age':18}
d = dict(name='py', age=18)
# 字典推导式
{x:x**2 for x in range(3)}  # {0:0,1:1,2:4}
```

### 常用操作

```python
d['name'] = 'python'  # 赋值/修改
del d['age']          # 删除键值对
'name' in d           # 键存在判断 → True
list(d)               # 获取键列表 → ['name']
```

## 六、循环技巧

1. 字典遍历键值：`for k,v in d.items()`
2. 带索引遍历：`for i,v in enumerate(list)`
3. 多序列并行遍历：`for x,y in zip(list1,list2)`
4. 反向遍历：`for x in reversed(list)`
5. 排序遍历：`for x in sorted(list)`
6. 去重遍历：`for x in sorted(set(list))`

## 七、条件与序列比较

- 比较运算符：`in / not in / is / is not`
- 布尔运算：`and`（短路）、`or`（短路）、`not`
- 序列比较：按字典序逐元素对比，短序列更小
