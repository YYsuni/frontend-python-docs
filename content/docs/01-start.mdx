## 一、启动 Python 解释器

1. **各系统启动命令**

- Unix/Linux：终端输 `python3.12`
- Windows：`python3.12` 或 `py` 启动器
- 退出：Unix `Ctrl+D` / Windows `Ctrl+Z`，通用 `quit()`

2. **3种常用启动方式**

- 交互式：直接输 `python3`，逐行执行代码
- 命令行执行代码：`python3 -c "代码语句"`（语句建议加引号）
- 模块当脚本运行：`python3 -m 模块名`
- 脚本跑完进入交互：`python3 -i 脚本.py`

3. **行编辑能力**
   依赖 `GNU Readline`，支持代码补全、历史命令、光标编辑；`Ctrl+P` 可快速检测是否支持。

## 二、命令行参数 sys.argv

1. 所有命令行参数以**字符串列表**存在 `sys.argv`
2. 规则：

- 无脚本无参数：`sys.argv[0]` 为空串
- `python -c` 模式：`sys.argv[0] = '-c'`
- `python -m 模块` 模式：`sys.argv[0]` 为模块完整路径名
- `-c`/`-m` 后的参数全部保留在 `sys.argv`，由业务代码自己解析

## 三、交互模式

1. 提示符：`>>>` 主提示符、`...` 多行次提示符
2. 适合临时测试小段代码、逻辑片段，和浏览器 Console / Node REPL 逻辑一致

## 四、源码文件编码规则

1. **默认编码**：**UTF-8**，支持中文、多语言字符
2. 自定义编码声明写法：

```python
# -*- coding: cp1252 -*-
```

3. 若文件首行是 shebang `#!/usr/bin/env python3`，编码声明放**第二行**即可。
