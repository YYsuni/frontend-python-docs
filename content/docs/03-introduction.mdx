## 一、基础规则

1. 注释：`#` 单行注释，字符串内`#`不生效
2. 交互终端：`>>>` 输入，无提示符是输出；多行代码空行结束
3. 代码靠**缩进**分块，无`{}`，缩进必须统一

## 二、数字运算

1. 基础运算符：`+ - * /`
   - `/` 永远返回**浮点数**
   - `//` 向下取整除法、`%` 取余、`**` 幂运算
2. 类型：`int`整数 / `float`浮点数；混合运算自动转浮点
3. 变量赋值：`=` 赋值，未定义变量直接用抛 `NameError`
4. 交互特有：`_` 自动接收上一次表达式结果（只读）
5. 支持复数：后缀 `j` 表虚数；还有 Decimal/Fraction 高精度类型
6. 优先级：`**` 高于负号，`-3**2 = -9`，需加括号 `(-3)**2`

## 三、字符串 str

1. 声明：单引号`''` / 双引号`""` 等效
2. 转义：`\` 转义；`r''` 原始字符串，不转义（适合文件路径）
3. 多行字符串：`"""..."""` / `'''...'''`
4. 拼接
   - 字面量紧邻自动拼接：`'Py' 'thon'`
   - 变量拼接只能用 `+`
   - `*` 重复字符串：`3 * 'ab'`
5. 索引 & 切片（和TS数组逻辑类似）
   - 正向从0开始，反向 `-1` 最后一位
   - 切片 `s[start:end]` **含头不含尾**，缺省从头/到尾
   - 索引越界报错，切片越界自动容错
6. **不可变**：str 不能修改单个字符，只能新建字符串
7. 长度：`len(字符串)` 获取长度

## 四、列表 list

1. 定义：`[]` 逗号分隔，可存任意类型元素
2. 同字符串：支持**索引、切片、`+` 合并**
3. **可变**：可直接修改元素、支持 `append()` 尾部追加
4. 引用特性：列表赋值是**引用传递**，多个变量指向同一列表，改一个全变
5. 浅拷贝：`arr[:]` 生成新列表，互不影响
6. 切片赋值：可批量替换、删除、清空列表
7. 嵌套列表：支持多维嵌套，`x[0][1]` 取嵌套元素
8. 长度：`len(列表)`

## 五、编程入门基础

1. 多重赋值：`a, b = 0, 1` 同时解构赋值，右侧先全部求值再赋值
2. `while` 循环：条件为真持续执行，靠**缩进**划分循环体
3. 比较运算符：`< > == <= >= !=`
4. `print()` 打印
   - 多参数自动空格分隔
   - `end=','` 自定义结尾，取消默认换行
